Decisões Técnicas

Streamlit

Escolhido pela rapidez de prototipagem e facilidade de criar UI interativa sem frontend complexo.

Plotly

Biblioteca para gráficos interativos (zoom/hover), incluindo composições (barras + linhas) e responsividade.

Cache (st.cache_data)

Evita recarregar o CSV a cada interação, melhorando performance e fluidez na navegação.

Agregação Mensal

Datas convertidas para períodos mensais, simplificando séries temporais e destacando tendências sem ruído diário.

Programação Defensiva

Verificações de existência do dataset e mensagens amigáveis orientam o usuário no setup (ex.: arquivo ausente).
